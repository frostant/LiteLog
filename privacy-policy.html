隐私政策

最后更新日期：2025年12月20日

概述

热量追踪（以下简称"本应用"）非常重视用户的隐私保护。本隐私政策说明了我们如何收集、使用和保护您的信息。

信息收集

本应用收集和使用以下信息：

  • 食物照片：用于AI识别食物种类和热量。照片仅用于识别，不会上传到我们的服务器。
  • 饮食记录：包括食物名称、热量、营养成分等，存储在您的设备本地。
  • API Key：您自行提供的OpenAI API Key，加密存储在设备本地。

信息使用

我们使用收集的信息用于：

  • 提供食物识别和热量计算服务
  • 记录和展示您的饮食历史
  • 改进应用功能和用户体验

第三方服务

本应用使用 OpenAI API 进行食物识别。当您使用识别功能时，食物照片会发送至 OpenAI 服务器进行处理。请参阅 OpenAI 隐私政策 了解更多信息。

数据存储

所有个人数据（包括饮食记录、API Key）均存储在您的设备本地，使用 iOS 系统提供的安全存储机制。我们不会将您的数据上传到我们的服务器。

数据安全

我们采取以下措施保护您的数据：

  • API Key 使用 iOS Keychain 加密存储
  • 所有数据仅存储在您的设备上
  • 不与任何第三方共享您的个人数据

您的权利

您可以随时：

  • 在应用内删除您的饮食记录
  • 删除或更换您的 API Key
  • 卸载应用以删除所有本地数据

儿童隐私

本应用不面向13岁以下儿童。我们不会故意收集13岁以下儿童的个人信息。

隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在应用内或本页面公布。
